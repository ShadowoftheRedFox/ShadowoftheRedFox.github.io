About the game
Sur le jeu

It's a project I'm working on for nearly 3 years now. I just wanted to make a little game from scratch by myself.

C'est un projet où j'y travaille depuis presque 3 ans maintenant. Je voulais faire juste un petit jeu à partir de rien.

It's a 2D RPG using HTML/CSS/JS. And I'll add in this game any fantasy I have, though I'm open for suggestions.

C'est un JDR en 2D en utilisant HTML/CSS/JS. Et j'ajouterai dans ce jeu toutes les fantaisies que j'ai, même si je suis ouvert au suggestions.

I keep the code open source, and most of my projects linked to this website are public too (such as the minigames).

Je garde le code source publique, et la plupart des projets liés à cette page le sont aussi (comme les minijeux).
